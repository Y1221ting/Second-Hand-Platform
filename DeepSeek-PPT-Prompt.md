# DeepSeek Prompt — 竞赛答辩PPT生成

> 复制以下内容，直接粘贴到 DeepSeek (chat.deepseek.com)
> 它会自动生成10页PPT的详细内容

---

## Prompt（直接复制）

```
请为我生成一份学科竞赛答辩PPT的详细内容，共10页。

【项目信息】
项目名称：校园二手交易平台（Second-Hand）
面向用户：江西省38所高校学生
技术栈：React 18 + Tailwind CSS 前端，Node.js + Express 后端，MongoDB 数据库，通义千问AI
部署方式：Docker Compose容器化，2核2G云服务器，域名 freevian.top
核心功能：购物车系统、38校模糊搜索、AI智能推荐、原子操作防超卖、手机号正则校验

【团队信息】
学校：南昌师范学院
学院：数学与信息科学学院
专业：24数据科学与大数据技术本科班
队员：杨婷（队长）、曾富、阳汩清
指导老师：梅能

【答辩时长】8分钟（含4分钟现场演示）

【格式要求】
请按照以下结构生成10页PPT内容：
第1页：封面（项目名称+团队完整信息）
第2页：目录
第3页：项目背景（包含数据：78%学生有二手需求、65%担心安全、52%认为运费高、43%希望同校交易）
第4页：技术方案（React+Node+MongoDB+AI四层架构，Docker部署）
第5页：功能实现（用户系统、商品管理、购物车、搜索、推荐、安全机制）
第6页：项目创新点（AI推荐系统、38校生态、轻量级架构、电商闭环四个角度）
第7页：应用与推广（社会/经济/安全价值，三阶段推广路线）
第8页：现场演示（写明域名 freevian.top，按时间线列出演示步骤）
第9页：未来展望（短/中/长期规划）
第10页：致谢（完整团队信息+联系方式）

【注意事项】
- 每页先写标题，冒号后写正文要点
- 正文用短句或要点列表，便于放在PPT上
- 语言正式但简洁，适合评委阅读
- 第8页演示环节要写明每步的操作和讲解词
- 封面和致谢页要包含完整团队信息
- 每页底部标注：南昌师范学院 · 数学与信息科学学院 · 24数据科学与大数据技术本科班
```

---

## 使用步骤

```
① 打开 https://chat.deepseek.com
② 粘贴上面的整个 Prompt 到对话框 → 按 Enter
③ DeepSeek 会生成约10页PPT内容
④ 检查内容，可以要求修改某一页
⑤ 复制生成的内容 → 粘贴到 ChatPPT (chatppt.cn)
⑥ 选择「学术蓝」或「极简白」模板 → 生成PPT
⑦ 下载 PPTX 格式 → 用 PowerPoint 微调排版
```

---

## 可选：修改某一页

如果 DeepSeek 生成后你想调整某一页，直接说：

```
第X页重新写，重点突出XXX
```

例如：
- "第4页技术方案再精简一些，只保留架构图"
- "第6页创新点每个点加粗关键词"
- "第8页演示脚本增加一个AI推荐的操作步骤"

---

## 备用：ChatPPT 直接导入

如果你不想经过 DeepSeek，也可以直接把项目根目录下的 `PPT内容精简版.md` 文件内容，**全选复制 → 粘贴到 ChatPPT 的输入框**，选择模板直接生成。
