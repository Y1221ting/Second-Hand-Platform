## 优化优先级

### P0（最影响分数，改了立竿见影）

1. **登录页渲染阻塞 5,570ms** → 检查登录页加载了什么大的 CSS/JS，加 defer 或拆包
2. **图片未优化** → Nginx 开启图片压缩、WebP 支持或缩略图方案
3. **商品详情 CLS 0.142** → 商品图片标签加 width/height 固定宽高

### P1（优化后可以提升）

4. **渲染阻塞请求缩减** → 关键 CSS 内联，非关键资源加 defer
5. **缓存策略** → Nginx 配置静态资源 30 天强缓存

### P2（锦上添花）

6. 减少未使用 JS/CSS → 路由懒加载
7. 无障碍修复 → 加 label、main 标签、颜色对比度调整
