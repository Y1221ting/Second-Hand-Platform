### 登录修复（重要）
- **根因**：2GB 服务器上原生 bcrypt (C++ 扩展) 在 verifyPassword 时触发 OOM Kill，进程被静默杀掉
- **修复**：`auth.js` 中 `require("bcrypt")` → `require("bcryptjs")`（纯 JS 实现，内存安全）
- 盐轮数从 12 降为 8（减少计算内存占用）
- Docker 容器内 `npm install bcryptjs`
