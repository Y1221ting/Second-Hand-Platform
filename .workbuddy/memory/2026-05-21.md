# 2026-05-21 工作日志

## 修复：addToCart 加入购物车数量校验缺失

**问题描述：**
- `addToCart`（POST /:productId）不读取 `req.body.quantity`，传递的负值、超大值都被静默忽略，永远只加1件，始终返回200

**修复内容：**
1. 从 `req.body` 读取可选 `quantity` 参数，默认值为1
2. 增加正整数校验：`!Number.isInteger(quantity) || quantity < 1` → 400 "数量必须为正整数"
3. 增加库存上限检查：`quantity > product.quantity` → 400 "库存不足，当前库存 N 件"
4. 购物车已有该商品时，合并数量不超过库存上限

**涉及文件：**
- `Server/controllers/cartController.js` — addToCart 方法
