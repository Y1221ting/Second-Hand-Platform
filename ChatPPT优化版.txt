项目背景

现状分析：
江西省高校在校生超过120万人，毕业生离校产生大量闲置物品
新生入学季消费集中，二手商品需求旺盛
现有主流平台缺乏校园场景化支持

痛点调研数据：
78% 有二手交易需求但缺乏合适渠道
65% 担心校外平台交易安全问题
52% 认为运费成本高于商品本身价值
43% 希望能在同校内进行交易

解决方案：
搭建面向江西省38所高校、支持同校认证、校内流转、面对面交易的校园专属二手交易平台

---

技术方案

系统架构：
用户端 — React 18 + Tailwind CSS
反向代理 — Nginx 端口5000
后端服务 — Node.js + Express 4
数据库 — MongoDB 7.0 Docker容器化
AI引擎 — 通义千问 Qwen-plus 大模型

部署架构：
前端 — Nginx静态文件，反向代理/api/到后端
后端 — Node.js + Express 4，RESTful API
数据库 — MongoDB 7.0，Docker运行
部署 — Docker Compose三容器一键启动
服务器 — 2核2G云服务器，域名freevian.top

AI集成方案：
商品描述智能生成
智能推荐系统
商品分类推荐

功能实现

核心功能：
用户系统 — 注册登录JWT认证，bcryptjs加密
商品管理 — 发布编辑删除，AI辅助描述
商品列表 — 38校模糊搜索，多维度筛选排序
商品详情 — 多图展示，原子操作防超卖
购物车 — 添加修改删除，批量原子结算
智能推荐 — AI推理标签，五级漏斗引擎
个人中心 — 发布记录购买记录购物车管理

安全机制：
bcryptjs加密，8轮盐值
JWT令牌身份认证
手机号正则校验
地址长度校验
价格上限控制
原子操作防超卖
NoSQL注入防护
